# ブロック崩し / Breakout Game

HTML5 Canvas + JavaScript で作られたブロック崩しゲームです。

## 遊び方

**PC**:
- 矢印キー（← →）またはマウス移動でパドルを操作
- Enter キーまたはボタンでゲーム開始

**スマートフォン**:
- タッチ操作でパドルを操作
- 画面タッチでゲーム開始

## ゲーム内容

- 5レベル（レベルが上がるほどボール速度アップ）
- 6段 × 10列のカラーブロック
- スコア・ライフ・レベル表示
- ゲームオーバー / クリア画面
- レスポンシブ対応（PC・スマホ両方で動作）

## 技術仕様

- HTML5 Canvas
- Vanilla JavaScript（フレームワーク不使用）
- サーバー不要（静的ファイルのみ）
